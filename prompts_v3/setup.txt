# V3 Setup Instructions - Session Continuity System

## Quick Setup (Recommended)

### Replace V2 System
```bash
# Backup current system
cp ~/.claude/CLAUDE.md ~/.claude/CLAUDE-v2-backup.md

# Install V3 core system
cp ./prompts_v3/core-rules.md ~/.claude/CLAUDE.md

# Create context directory for session continuity
mkdir -p ~/.claude/docs
mkdir -p ~/.claude/archive

# Optional: Install full intelligence
mkdir -p ~/.claude/commands/mode-details
cp ./prompts_v3/mode-details/* ~/.claude/commands/mode-details/
```

### Verify Installation
```bash
# Check core system
ls -la ~/.claude/CLAUDE.md

# Check mode details (optional)  
ls -la ~/.claude/commands/mode-details/

# Check session directories
ls -la ~/.claude/docs/
ls -la ~/.claude/archive/
```
